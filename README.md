## ‚öôÔ∏è Quick Setup

After installation, run your first RustAPI project:

1. Open your terminal or command prompt.
2. Navigate to your project directory.
3. Type `rustapi start myproject` to create a new project.
4. Change into your project directory with `cd myproject`.
5. Start the server by typing `rustapi run`.

Now visit `http://localhost:8000` in your web browser. You should see your new RustAPI application running!
